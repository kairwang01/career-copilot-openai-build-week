2. Review the candidate resume report, career workspace, matching, application tracking, and interview preparation.
3. Open the employer experience to inspect structured candidate matching and hiring stages.
4. Use the repository README and linked evidence to inspect architecture, governed AI execution, tests, and release boundaries.

The public demo is the fastest evaluation path. Local installation remains available for source-level testing.
